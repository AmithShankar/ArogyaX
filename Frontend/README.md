# ArogyaX - Frontend

Modern clinical dashboard built with Next.js and pnpm.

### 🌐 Live Demo
<a href="https://arogyax.amithshankar.in" target="_blank"><img src="https://img.shields.io/badge/Visit-ArogyaX-blue?style=for-the-badge&logo=nextdotjs" alt="Live Demo"></a>

### 🚀 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State/Auth**: React Context + Axios
- **Iconography**: Lucide React

### 📦 Installation

1.  **Install Dependencies**:
    ```bash
    pnpm install
    ```

2.  **Environment Variables**:
    Configure your `.env` by referring to [`.env.example`](.env.example).

3.  **Run Development Server**:
    ```bash
    pnpm dev
    ```

### 🛠️ Key Features
- **Dynamic Charting**: Medical observations and vital tracking.
- **Admin Management**: Dedicated staff and department control.
- **Responsive Workflows**: Optimized for both desktop and tablet views.
- **One-Click Demo Helper**: Quick access for local testing and review.
